# ¿Qué son los packages?

> Son agregados que te van a simplificar el desarrollo y gestíon de proyectos

## ¿Cómo se instalan?
 
> Primero necesitamos el package control
> Para instalar el package control debemos ir a:   
    Tools -> Install package Control  
> Esto instala el package control

## ¿Qué packages me conviene instalar?

> Uno que completamente necesario es el Emmet

> Hay que ir a Package Control (que está en preferencies)
> Cuando llegaste ahí, te da una serie de acciones
> De esas acciones vamos a ir a 
    Install package

> una vez instalado ya lo podemos usar.
> referencia de usp y documentación

    https://docs.emmet.io/abbreviations/syntax/
